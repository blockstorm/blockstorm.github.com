About Blockstorm

  • Hardware

    Blockstorm is hosted on a dual-core server with 2GB of RAM, SSD storage and a 1 GB/sec network interface.

  • Griefing protection

    Using LogBlock we can log every player's actions which makes it easy to find griefers and undo any damage.

  • Economy

    The /shop command and chest shops allow players to sell and buy items.
    Find out how to use shops.

  • Getting around

    Additionally to the portals and [teleporter] signs at spawn every player can set up to ten homes.

  • Cheating prevention

    With NoCheatPlus we can detect and prevent cheating.
    Read our rules.

  • Dynmap

    The dynmap plugin provides live maps of all worlds on the server.
    Access the dynmap.

  • LWC

    Using the LWC plugin players can decide who has access to their doors, chests and furnaces.

  • Custom plugins

    Several custom plugins were written specifically for Blockstorm.
    Read about commands.

  • Since 2011

    Blockstorm has been online for more than three years (since August 2011) with nearly 100% uptime.
